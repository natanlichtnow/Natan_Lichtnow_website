Oficina de Parapente project

A system developed to solve a major issue we have in Brazil: fake maintenance reports for paragliders

As request from the client, he wanted to digitalize his maintenance process, ensuring that there is an authenticated admin page for generation of the reports.

The report generation is dynamic and easy to use, with multiple added functions for quality of life and security of the process, it also allows his clients to check the progress in real time of how their maintenance is going.

Client asked to keep the GitHub repo private so no one can copy or change the system
